Declaration

init(value: [UInt8], paddingLength: Int)

Parameters

value

bytes of the bitstring in little endian order, i.e., least significant byte first. Any padding bytes should be prepended, i.e., padding bits begin in the leading bit of the first byte.

paddingLength

Length of padding.
